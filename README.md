# Folly Flow 🌊

**Live Crowd + Traffic Meter for Folly Beach, SC**

Real-time conditions so you know if it's worth driving down.

### ✨ Live Demo
**[Open Folly Flow](./index.html)** (just open the file in your browser)

### Features
- Beautiful ocean-themed premium design
- Animated live crowd meter
- Real-time traffic status on Folly Road
- Smooth refresh button
- Mobile-friendly + installable as app
- Works offline

### Why It Exists
After the July 4th chaos, I wanted to know instantly if Folly was still packed or finally chill.

### How to Use
1. Open `index.html` in any browser
2. Bookmark it or "Add to Home Screen" on your phone
3. Hit Refresh for latest conditions
